# WINCS02 & RNWF02 Device Variants

**WINCS02** and **RNWF02** are available as modules with these variants:
- Either an on-board PCB antenna or a U.FL connector for an external antenna
- Either with or without an integrated Microchip Trust&Go CryptoAuthentication device; if present, this can be accessed from the host via an I2C interface

**WINCS02** is also available as IC-only.

**WINCS02** and **RNWF02** add-on boards are available for evaluation and development purposes.
