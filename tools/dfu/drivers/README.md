# FTDI D2XX Drivers

The DFU tool requires FTDI D2XX drivers to communicate with the USB-to-UART
bridge cable.

## Download

Download the driver for your operating system (Windows, Linux, macOS) from FTDI:

https://ftdichip.com/drivers/d2xx-drivers/

## Recommended Cable

**FTDI TTL-232R-3V3** — 3.3 V USB-to-UART cable with 6-pin header.

Ensure the cable is 3.3 V (not 5 V) to match the RNWF02/WINCS02 I/O voltage.
