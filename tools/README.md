# Tools

Supporting tools for **WINCS02** and **RNWF02** firmware and file management.

| Tool | Description |
|------|-------------|
| [atfs/](atfs/) | File System Utility tool — Manage files on a **RNWF02** device via AT commands |
| [cfgc/](cfgc/) | Configuration Command Compiler — Compile, encode, and decode configuration command files |
| [dfu/](dfu/) | Device Firmware Update tool — Program or update firmware on a non-running device via FTDI cable |
| [flfs/](flfs/) | Flash File System tool — Build custom file system images with TLS certificates, keys, and configuration |
| [nvm_update/](nvm_update/) | NVM Update tool — Update **RNWF02** firmware via AT commands |
